# Build Your Own Coding Agent Harness

A hands-on course on building a coding agent from scratch using Grok.

## Course Overview

Learn to build a coding agent harness step by step, understanding the core concepts that power modern AI coding assistants.

## Lessons

### 01 Minimal Agent Loop
Understand the foundation: the basic agent loop that takes user input, calls a model, and processes the response.

### 02 Tool Call
Learn how to define and execute tools. Tools are the bridge between the agent's reasoning and real-world actions.

### 03 Read/Edit/Execute
Implement file operations: reading source code, editing files, and executing commands to test your agent.

### 04 Permissions
Build a permission system to control what operations the agent can perform, ensuring safe execution.

### 05 Compaction
Learn to compress conversation history to keep the context window manageable for long sessions.

### 06 Checkpoint
Implement checkpointing to save and restore agent state, enabling pause and resume functionality.

### 07 Subagent
Create a subagent architecture to delegate specialized tasks to child agents for modular, scalable design.

### 08 MCP
Integrate the Model Context Protocol (MCP) to connect your agent with external tools and data sources.

## Getting Started

Each lesson builds on the previous one. Start with lesson 01 and work your way through to build a complete coding agent harness.
